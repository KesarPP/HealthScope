## 🩺 HealthScope – AI-Powered Personal Health Companion

HealthScope is an intelligent health assistant web application that helps users monitor, compare, and improve their health through AI-powered analysis, natural remedies, diet planning, and personalized recommendations.

## 🚀 Features

- 🔐 **Secure Login System** (Firebase Auth)
- 🏠 **Dashboard** with easy navigation
- 📊 **Compare Health Reports** (PDF or JSON)
- 🧠 **AI Chatbot** (Gemini 1.5 Pro) for personalized insights
- 🥗 **Smart Diet Recommendation** based on:
  - Age group
  - Blood type
  - Allergies
  - Diseases & Medications
- 🧘 **Yoga and Natural Health Suggestions**
- 📅 **Appointment Manager** with notification reminders (via FCM)
